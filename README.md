
# Data Cleaning and Exploratory Data Analysis (EDA) using Python

## Project Overview

This project demonstrates the complete data cleaning process using Python and Pandas in Google Colab.

The objective of this project was to clean a real-world dataset, handle missing values, prepare the data for analysis, and perform basic exploratory data analysis (EDA).

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Data Cleaning Steps

- Imported the dataset
- Inspected the dataset using:
  - head()
  - info()
  - describe()
- Checked missing values
- Removed unnecessary columns
- Filled missing numeric values using the Median
- Filled missing categorical values with "Unknown"
- Converted date columns to datetime format
- Cleaned column names by removing extra spaces
- Verified data types

---

## Exploratory Data Analysis (EDA)

- Generated descriptive statistics
- Performed correlation analysis
- Visualized relationships using a correlation heatmap
